Moderation Skript

Simple, powerful tools for server staff

About

The Moderation Skript is a lightweight, single file moderation toolkit designed for SMPs, PvP servers, and community servers. It gives staff clean, fast, and reliable commands without needing a full plugin suite.

Features

  • Mute / unmute players
  • Kick / ban / tempban commands
  • Warn system with logs
  • Staff chat toggle
  • Freeze players for investigation
  • Clear chat (global or local)
  • View punishments for any player
  • Silent punishments for staff
  • IP-ban support (optional)
  • Customizable messages and formatting
