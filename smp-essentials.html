SMP Essentials Skript

A single file, fully configurable SMP control system

About

SMP Essentials is a powerful all in one Skript designed to give server owners complete control over gameplay rules, enchant limits, PvP settings, grace periods, world behavior, and more all from a clean and simple /smp menu.

Features

  • Enchantment caps
  • PvP toggle + grace period timer
  • Totem disable option
  • Pearl toggle
  • Mace crafting limits + cooldown editor
  • Potion effect limits
  • Creeper explosion control
  • 1-player sleep
  • Villager restock toggle
  • Player head drops
  • Netherite recipe toggles
